# Hero Video

Drop your `orbitpay-hero.mp4` file here.

The video should be:
- 16:9 aspect ratio (1920x1080 or 1280x720)
- MP4 + H.264 codec + AAC audio
- Under 10 MB
- 10-30 seconds long
- Loops seamlessly (it auto-plays muted on the landing page)

The landing page references it as: `/assets/videos/orbitpay-hero.mp4`

Until you upload the file, the landing page falls back to a static banner image.
